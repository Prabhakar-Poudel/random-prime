# random-prime

> Generate a random prime number

## Install
```
npm i random-prime --save
```

## Example
```javascript
import randomPrime from 'random-prime';

console.log(randomPrime());
// 254205915209711
console.log(randomPrime(500));
// 119
console.log(randomPrime(200, 800));
// 413
```
